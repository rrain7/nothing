请实现一个函数，输入一个整数，输出该数二进制表示中 1 的个数。例如，把 9 表示成二进制是 1001，有 2 位是 1。因此，如果输入 9，则该函数输出 2。

```go
func hammingWeight(num uint32) int {
   var cnt int
   for num > 0 {
      if num % 2 == 1{
         cnt++
      }
      num >>= 1
   }
   return cnt
}
```
